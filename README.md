# react-native-jsi-view-helpers

React Native Jsi view helpers

## Installation

```sh
npm install react-native-jsi-view-helpers
```

## Usage

```js
import { multiply } from "react-native-jsi-view-helpers";

// ...

const result = await multiply(3, 7);
```
